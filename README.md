## Ambiente de desenvolvimento

Este projeto usa Python em ambiente virtual local (`.venv`).

### 1) Criar e ativar o ambiente virtual

```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 2) Instalar dependências

```bash
pip install -r requirements.txt
```
